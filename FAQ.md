## ❓ Häufige Fragen (FAQ)
---

Server starten nicht? 
1. Prüfe, ob du Java installiert hast. Ohne Java kann die `paper.jar` nicht ausgeführt werden
2. Prüfe, ob du die richtige Java version installiert ist! 
- die richtige java verion: 
Für die neuesten Minecraft-Versionen empfehle ich Java 21. 
Du kannst es dir kostenlos bei [Adoptium (Temurin)](https://adoptium.net/de) herunterladen. 
Achte darauf, die "JRE" oder das "JDK" für dein Betriebssystem (Windows x64) zu wählen.
